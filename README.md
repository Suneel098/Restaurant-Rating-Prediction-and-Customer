# Restaurant-Rating-Prediction-and-Customer
Restaurant Rating Prediction is a Machine Learning project that predicts restaurant ratings using Python, Pandas, NumPy, Seaborn, and Scikit-learn. The project includes data cleaning, visualization, heatmaps, EDA, and ML models like Linear Regression and Random Forest to analyze customer and restaurant data.
# 🍽️ Restaurant Rating Prediction and Customer Analysis

A Data Science project focused on exploring restaurant data, analyzing customer preferences, and identifying the factors that influence restaurant ratings. This project uses Python for data preprocessing, visualization, feature engineering, and business insights.

---

# 📌 Project Overview

The objective of this project is to perform Exploratory Data Analysis (EDA) on restaurant data to discover meaningful insights about customer behavior, restaurant ratings, pricing, online delivery, table booking, cuisines, and locations.

The project demonstrates practical data analysis skills using Python and popular data science libraries.
